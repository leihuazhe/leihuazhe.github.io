我们先了解一下一些必要的信息。ajax 请求和普通的 http 请求是不一样的，Ajax请求是XMLHTTPRequest对象发起的，而http请求是浏览器发起的。﻿

二者不同地方体现在HTTP请求的头信息中。﻿

AJAX请求头中带有X-Requested-With信息，其值为XMLHttpRequest。而普通请求是没有的。﻿
